# Konachan爬虫

爬取K站图片  
新手练手项目，欢迎大佬PR指正！  
利用``tags``，按照月份进行爬取图片，自动爬取**原图**  
多线程下载（不会写异步...）  
错误记录会以``JSON``格式保存在下载目录的``error.json``文件中，方便查看错误及重新下载  

效果图请查看：  
博客：<https://blog.sukap.cn/posts/6/>

适用于：  
 <https://konachan.net> 和 <https://konachan.com>
